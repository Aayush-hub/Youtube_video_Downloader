# Youtube_video_Downloader
### Pre-requisites:
    pip install pytube or $ git clone git://github.com/nficano/pytube.git
    pip install tqdm

- Fork this repository.
- Clone the repo.
- Open in any IDE.
- Run the program and enjoy video.
